# **Game Shark - Pokermon**
Pokermon add-on with new "cheaty" Jokers!

Focusing on non-Pokemon Jokers that expand the scope of gameplay.
One goal is to test concepts to potentially add to the main mod. Please give feedback in the Pokermon discord!

## Jokers

Each Joker can be toggled on/off in the mod config options:

| | | |
|-----|------|-----|
| Dark Cave | Common | Find Uncommons with Flash
| Celadon Game Corner | Rare | Let's go gambling with 7s

## UI Customization

Choose the following UI options:

| | | |
|-----|------|-----|
| Poke Ball | Background color | Red and white |
| Ultra Ball | Background color | Yellow and white |
